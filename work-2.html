The Talking Graduation Cap
A Guessing Game Like No Other

Project Overview

Inspired by the Harry Potter Sorting Hat and the game Akinator, this project aimed to create an interactive graduation cap that can guess which Dyson School staff member the user is thinking of, using only audio questions and head movements (nodding for "yes", shaking for "no").

We designed a cap equipped with sensors, motors, and a speaker, all coordinated through serial communication between Arduino and Bela. The system follows an interactive flowchart based on the user's answers. While asking questions, the capâ€™s mouth moves in sync with the audio, and the final response triggers a dramatic or joyful tassel spin to conclude the interaction.

This project brings together electronics, mechanics, and interaction design in a playful and expressive way.
